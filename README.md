# COMMUNITY_TRACKING_SYSTEM
Welcome Guys!!!
This is a simple web app based on core concepts of pure J2EE (Java Enterprise Edition).
It is a web app on how authorities and common peoples can connect each other in order to colve any general issues such as water logging, street light repair etc.
I have use concepts of servlets, java server pages etc.
Moreover, I have integrated it with Maven, so you also need basic understanding about it as you can see pom.xml is used.
Also, this project follows the strict MVC structure i.e. MODEL-VIEW-CONTROLLER.
